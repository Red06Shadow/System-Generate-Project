# Comandos
1. -v/--version : Version del activo
2. ... -h/--help : Ayuda de comandos generales o si es uno especifico
3. -d/--destine "Ruta de localozacion" : Ingresa donde se generara el proyecto, por defecto la ruta actual del terminal
4. -n/--name "Nombre del projecto" : Ingresa el nombre del proyecto(Tanto para carpetas como para valores internos(proximamante)), por defecto el nombre de la carpeta acutual de la ruta de la terminal
5. -f/format-- : Formato del proyecto, obligatorio, define para que lenguaje y 
como se generara el proyecto
5. 1. Si se agrega -o como comando se permite leer un archivo de configuracion definido por el usuario
5. 2. Si no, se leera simplemente la cadena como el nombre de un archivo dentro del repositorio del programa
